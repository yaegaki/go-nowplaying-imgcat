# go-nowplaying-imgcat
Nowplaying with image.

# Install
```sh
go get -u github.com/yaegaki/go-nowplaying-imgcat
```

# Usage 
Only support iTerm2 Ver 3.
```sh
cd $GOPATH/src/yaegaki/go-nowplaying-imgcat
go run main.go
```
